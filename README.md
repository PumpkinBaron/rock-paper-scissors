# rock-paper-scissors
Rock paper scissors console program in javascript for The Odin Project

This simulates rock paper scissors between a human and an AI. The AI chooses
rock paper or scissors randomly, while the human inputs a string to choose from
the same set. They play five rounds, at which point the one who won the most 
rounds wins.

A GUI will eventually be added, but, to start, it will be a console program.
